LIBREWALLET — instalacja
========================

Język: Polski (ten plik) · English (INSTALL.en.txt)

Pobierz najnowszą wersję:
https://github.com/wedishprocentahc/librewallet/releases


macOS — Apple Silicon (M1 / M2 / M3 / M4)
-----------------------------------------

Plik: LibreWallet-1.1.6-mac-arm64.pkg

KROK 1 — Sprawdź Mac
  Menu  → Informacje o tym Macu → Chip: Apple M1/M2/M3/M4

KROK 2 — Pobierz
  Releases → najnowsza wersja → Assets → LibreWallet-1.1.6-mac-arm64.pkg
  Plik ląduje w folderze Pobrane.

KROK 3 — Uruchom instalator (macOS może zablokować!)
  LibreWallet nie jest z App Store — zwykły dwuklik często NIE działa.

  Najłatwiej:
    Finder → Pobrane → PRAWY przycisk na pliku .pkg → Otwórz → Otwórz

  Jeśli nadal blokada:
    1. Spróbuj dwukliknąć .pkg (pojawi się komunikat o blokadzie)
    2. Ustawienia systemowe → Prywatność i ochrona
    3. Na dole: Bezpieczeństwo → „Otwórz mimo to” przy LibreWallet
    4. Podaj hasło administratora
    5. Uruchom .pkg ponownie

KROK 4 — Instalacja
  Kontynuuj → Zainstaluj → czekaj na „Instalacja zakończona pomyślnie”

KROK 5 — Język
  Wybierz Polski lub English (zmiana później: Import i ustawienia → Język)

KROK 6 — Uruchomienie
  Aplikacja: Finder → Aplikacje → LibreWallet
  Otwiera przeglądarkę: http://127.0.0.1:8787

  Jeśli macOS blokuje APLIKACJĘ (nie instalator):
    Aplikacje → PRAWY przycisk na LibreWallet → Otwórz → Otwórz
    LUB: Ustawienia systemowe → Prywatność i ochrona → Otwórz mimo to


macOS — Intel
-------------
Plik: LibreWallet-*-mac-x64.pkg (jeśli dostępny w Releases)
Te same kroki co powyżej (prawy przycisk → Otwórz, ewentualnie Ustawienia systemowe).


Windows
-------
1. Pobierz LibreWallet-*-win.exe z Releases (jeśli dostępny).
2. Uruchom dwuklikiem.
3. Otworzy się przeglądarka z aplikacją.


Import operacji
---------------
- XTB: przycisk Import → wybierz import XTB → plik .zip z eksportu platformy
- Inne źródła: przycisk Import → import uniwersalny → pobierz szablon CSV i przygotuj plik


Twoje dane
----------
Wszystko zostaje w przeglądarce na tym komputerze (localStorage).
Kopia zapasowa JSON: Import i ustawienia → Kopia zapasowa.

Pomoc
-----
Aplikacja: http://127.0.0.1:8787
Internet potrzebny głównie do aktualizacji cen.
