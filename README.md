> **Uwaga / Note:** Na Macu instalatora często nie da się otworzyć zwykłym dwuklikiem — użyj **prawego przycisku → Otwórz** lub **Ustawienia systemowe → Prywatność i ochrona → Otwórz mimo to**. Szczegóły w README w wybranym języku.
>
> On Mac, the installer often cannot be opened with a normal double-click — use **right-click → Open** or **System Settings → Privacy & Security → Open Anyway**. See the language README for full steps.
